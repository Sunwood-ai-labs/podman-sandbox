# 10 - Claude Meeting Bot

Claude Code を cron で1時間おきに起動して「定時会議」を開くボット。

## 概要

1時間おきに Claude が会議を開催し、エージェントチームを編成して
ミッションを遂行する議事録をマークダウンで出力します。

## ミッション

**Podman学習リポジトリの改善とサンプルコードの品質向上**

## エージェントチーム

会議で以下のロールからチームを編成：

| ロール | 担当 |
|--------|------|
| 🔍 リサーチャー | 調査・情報収集 |
| 💻 デベロッパー | 実装・コード変更 |
| 📝 ライター | ドキュメント作成 |
| 🧪 テスター | 動作確認・テスト |
| 🎨 デザイナー | UI/UX改善 |

## アジェンダ（ローテーション）

1. コードレビュー & チーム編成
2. 技術調査 & 調査チーム編成
3. ドキュメント改善 & 執筆チーム編成
4. テスト計画 & QAチーム編成
5. アイデア出し & 実装チーム提案
6. リファクタリング & チーム編成
7. セキュリティ確認 & チーム編成
8. パフォーマンス最適化 & チーム編成

## 使い方

```bash
cp .env.example .env
# .env に API キーを設定

./run.sh

# 議事録一覧
ls workspace/

# 最新議事録確認
cat workspace/meeting_$(date +%Y%m%d)*.md | head -50

# ログ確認
podman logs -f claude-meeting

# 停止
./cleanup.sh
```

## 出力

```
workspace/
├── meeting_20260324_0000.md  # 0時の会議
├── meeting_20260324_0100.md  # 1時の会議
├── meeting_20260324_0200.md  # 2時の会議
└── ...
```

各会議ごとに個別の議事録ファイルが生成されます。
