# 👥 Organization

## 組織図

```
        ┌─────────────┐
        │     CEO     │  戦略・組織運営
        │  (30分毎)   │  会議ルール改善
        └──────┬──────┘
               │
        ┌──────┴──────┐
        │     CTO     │  技術実務
        │  (30分毎)   │  コード品質管理
        └─────────────┘
```

## 役割定義

### CEO (Chief Executive Officer)
- **責務**: 組織全体の方向性決定
- **権限**: 会議ルール、戦略、フォーカスの変更
- **頻度**: 毎時30分

### CTO (Chief Technology Officer)
- **責務**: 技術的な実務遂行
- **権限**: コード変更、ドキュメント更新の提案
- **頻度**: 毎時00分

## チーム編成パターン

CTO会議で必要に応じて以下のロールをアサイン：

| ロール | 担当 |
|--------|------|
| 🔍 リサーチャー | 調査・情報収集 |
| 💻 デベロッパー | 実装・コード変更 |
| 📝 ライター | ドキュメント作成 |
| 🧪 テスター | 動作確認・テスト |
| 🔒 セキュリティ | セキュリティ監査 |
